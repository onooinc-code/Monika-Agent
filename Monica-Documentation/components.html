Component Library

The UI is built from a set of reusable, single-purpose React components. This approach promotes consistency, maintainability, and faster development.

Core Layout Components

  • App.tsx: The root component that assembles the main layout.
  • Header.tsx: A smart component that conditionally renders a dashboard or conversation-specific header.
  • ConversationList.tsx: The collapsible left sidebar for listing and searching chats.
  • BookmarkedMessagesPanel.tsx: The collapsible right sidebar for bookmarked messages.
  • StatusBar.tsx: The persistent footer that displays usage metrics.

Chat & Message Components

  • MessageList.tsx: Renders the list of messages for the active conversation.
  • MessageBubble.tsx: A complex component for rendering a single message, handling markdown, toolbars, and metadata.
  • MessageInput.tsx: The user input area with text and attachment capabilities.
  • PlanDisplay.tsx: A specialized component to display a multi-step plan from the AI Manager.

Modal Components

All modals share a consistent look and are managed by the useModalManager hook.

  • SettingsModal.tsx: Main application settings panel.
  • ConversationSettingsModal.tsx: Settings for the currently active conversation.
  • HistoryModal.tsx: Displays the AI-generated conversation summary.
  • CognitiveInspectorModal.tsx: The "Workflow Visualizer" for the AI's step-by-step process.
  • TeamGeneratorModal.tsx: A wizard for generating a new team of AI agents.
