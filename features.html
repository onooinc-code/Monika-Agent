Monica Project: Implemented Features

A comprehensive list of all capabilities built into the application.

Core Conversation Features

Full Conversation Lifecycle

Create, search, select, and delete conversations. All chats are persisted in a filterable sidebar.

Rich Message Rendering

Full GitHub-Flavored Markdown support, including code blocks with syntax highlighting.

Image Attachments

Attach images to prompts, which are displayed in the chat and processed by the AI model.

Secure HTML Preview

Safely render AI-generated HTML code in a sandboxed iframe to prevent security risks.

Message Metadata

Each message displays its timestamp, token count, and AI response time for full transparency.

Long Message Handling

Long AI responses are auto-summarized and collapsed to keep the chat clean and readable.

AI-Powered Intelligence

Multi-Agent Framework

Define and use multiple, distinct AI agents with unique personalities, skills, and knowledge.

Dynamic Conversation Modes

Switch between AI-driven planning, autonomous agent selection, and manual user control on the fly.

Moderated Multi-Agent Chat

Enable a mode where agents can discuss topics with each other, guided by an AI moderator and custom rules.

AI-Powered Actions

Regenerate AI responses, summarize complex messages, or have the AI rewrite your own prompts for you.

Long-Term Memory

AI extracts key facts from conversations to build a persistent memory, providing context across sessions.

AI Tool Use

Agents can use function calling to access tools like a calculator or a sandboxed web shell.

Developer & Power-User Tools

Workflow & Prompt Inspector

Visualize the AI's step-by-step cognitive process and inspect the exact prompts sent to the model.

Live Status & Metrics

A real-time status bar shows what the AI is currently doing and tracks API usage statistics.

AI Team Generator

A guided wizard that uses AI to generate a complete team of specialized agents from a simple prompt.

Advanced Configuration

Fine-tune agents, conversation settings, and application preferences through detailed settings modals.

Import / Export

Backup and restore all conversations and settings via a single JSON file.

MCP Server Manager

Interface with external processes via the "Monica Control Protocol" to give agents real-world capabilities.
